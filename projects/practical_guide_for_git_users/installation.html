安裝說明¶

由於目前Git, GitLab 已經是十分成熟的工具與系統，因此相關安裝說明請自行參閱以下連結。

  • Git 官方網站
  • GitLab 官方網站

此外，GitLab官方目前僅於Debian及Ubuntu系統環境內進行開發測試，因此推薦使用Debian或Ubuntu安裝GitLab。
